2. Token & Context Discipline

  • [ ] Caveman Mode: Token conservative by default. Escalates "caveman-ness" (short, blunt, efficient) as API limits/budgets approach. Source: caveman
  • [ ] 50% Compaction Trigger: Automatic context compaction at 50% window. Summarizes everything except the last 20 messages via cheaper agent. [Reference: Wang, Min, Zou 2026]
  • [ ] Anti-Bloat System Prompts: Lean, instruction-dense files with zero filler.
  • [ ] Grep-n Navigation: Strictly prohibited from cat-ing whole files. Uses grep -n and ranged reads only.
  • [ ] Native Shell Supremacy: Prefers native shell commands over high-level tool wrappers to minimize overhead.
  • [ ] Mine Context First: Always checks existing conversation context before triggering tool calls.
  • prefer less tool calls, tools calls are expensive. last resort.

3. Security & Injection Defense

  • [ ] Isolated Security Plane: Separate process for prompt injection detection.
