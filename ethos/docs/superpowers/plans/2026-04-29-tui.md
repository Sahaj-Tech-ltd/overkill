# Ethos TUI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build the Ethos TUI — a split-pane terminal chat interface with personality, sidebar, and overlay dialogs.

**Architecture:** Bubble Tea Elm architecture. Root `appModel` holds pages + overlays. Messages flow down: root → page → components. All internal packages wired through `App` struct. Test-first: write tests, dispatch sub-agents to implement.

**Tech Stack:** Go, Bubble Tea, Lip Gloss, Bubbles, Glamour

**Spec:** `docs/superpowers/specs/2026-04-29-tui-design.md`

func (a *Agent) SetModel(modelID string) {
	a.mu.Lock()

func (a *Agent) Model() string {
	a.mu.RLock()
	defer a.mu.RUnlock()
	return a.model
}

	a.mu.RLock()
	defer a.mu.RUnlock()
