Overkill is a terminal AI coding agent. 104+ LLM providers, 10 messaging gateways, 43 exclusives no other harness has, a personality engine that learns you, and zero telemetry. Ever.

```sh
curl -fsSL https://overkill.my/install | sh
```

Or with Go: `go install github.com/Sahaj-Tech-ltd/overkill@main`

---

### what it does

- **Smart routing** ‚ÄĒ picks the cheapest capable model per task. 104+ providers auto-discovered from models.dev.
- **Memory that works** ‚ÄĒ Postgres-backed. Remembers what you did three sessions ago.
- **Personality engine** ‚ÄĒ Memo the elephant thinks with you. Cooking mode. Movie quotes. Frustration detection. SOUL.md self-moulding.
- **10 gateways** ‚ÄĒ Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Mattermost, Feishu, iMessage, SMS.
- **3 Walls quality gates** ‚ÄĒ won't push auth changes at midnight. Won't ship without tests.
- **Fully open source** ‚ÄĒ MIT + Apache-2.0. No black box, no telemetry, no "contact sales."

### what it isn't

- A thin wrapper around an API. Every feature is built in Go, not prompt-engineered.
- Another Claude Code clone. Different architecture, different philosophy.
- Polite. It'll tell you when your idea is bad.
